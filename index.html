State Subtractor

Enter strings describing the initial set of states and the states to be subtracted. States should be entered using the characters "1","0", and "-", where "1" represents an option that is taken, "0" an option that is not taken, and "-" an option that can be in either state. All strings entered must be of the same length for sucessful computation.
